# Draft
A simple implementation of [Raft](https://raft.github.io/raft.pdf) consensus algorithm in Kotlin. This project is intended for educational purposes and to demonstrate the core concepts of the Raft algorithm. It is not optimized for production use.

## Getting Started
1. Clone the repository:
   ```bash
   git clone git@github.com:omniCoder77/DRaft.git
   ```
2. Navigate to the project directory:
   ```bash
   cd DRaft
   ```
3. Make sure docker is installed and running. Start project with:
   ```bash
   docker compose up
   ```
